1) Dans le fichier "DeviceOrientationService.cs", à la ligne 31, assurez-vous de configurer le ComPort adéquat lié à votre machine.
2) Le fichier JSON "PossibleArtsCollection" doit être placé sur votre bureau.
3) Servez-vous des codes QR présents dans le dossier "QR-Code" afin de tester le scan lié à notre application.
4) Exécutez la commande suivante pour voir les informations de la base de données : sqllocaldb info mssqllocaldb.
5) Si vous voyez que la base de données est arrêtée (STOPPED), alors exécutez la commande suivante pour la démarrer : sqllocaldb start mssqllocaldb.
6) Si vous rencontrez des problèmes anormaux avec votre machine liés à la base de données, voici une vidéo de secours qui montre l'intégralité de notre application : https://youtu.be/f4VNVLbyUeA
